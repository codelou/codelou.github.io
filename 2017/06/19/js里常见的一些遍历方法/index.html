js里常见的一些遍历方法

发表于 2017-06-19

js遍历方法归纳

前言

每种语言都有自己的遍历方法，每种语言都有好几种不同的遍历方法，今天我就将js中常见的遍历方法拿出来归纳一下，也方便自己的记忆。ps:今天之后不要说自己只会for循环了.

首先我们定义一个数组:

var arr =[‘1’,’test’,2,true,’123dfg’];

  1. for循环，需要知道数组的长度，才能遍历。(这个我就不多说了)
  2. forEach循环,循环数组中的每个元素来采取操作，没有返回值，可以不用知道数组的长度
var arr =['1','test',2,true,'123dfg'];
arr.forEach(function(i,index){
    i= i+',hi';
    console.log(i);
});
1,hi 
test,hi
2,hi
true,hi,
123dfg,hi
  1. map函数，遍历数组每个元素，并回调操作，需要返回值，返回值组成新的数组，原数组不变。
var arr=['1','test',2,true,'123dfg'];
var tt = arr.map((i) => {
  i = i+',hi2';
  return i;
});
console.log(arr,tt);
['1','test',2,true,'123dfg']
['1,hi2','test,hi2','2,hi2','true,hi2','123dfg,hi2']
  1. filter函数, 过滤通过条件的元素组成新的数组，原数组不变
var arr =['1','test',2,true,'123dfg'];
var tt = arr.filter((i) => {
 return typeof i =='number';
});
console.log(arr,tt);
['1','test',2,true,'123dfg']
[ 2 ]
  1. some 函数，遍历数组中是否有符合条件的元素，返回 Boolean值
var arr = ['1','test',2,true,'123dfg'];
var tt = arr.some((i)=>{
return typeof i =='number';

});
console.log(arr,tt);
['1','test',2,true,'123dfg'] true
  1. every 函数,遍历数组中是否每个元素都符合条件,返回Boolean值
    var arr = [‘1’,’test’,2,true,’123dfg’];
    var tt = arr.every((i) =>{
    return typeof i ==’number’;
    });
    console.log(arr,tt);
    [‘1’,’test’,2,true,’123dfg’]; false

当然除了遍历数组之外，还有遍历对象,常见方法in

var obj= {a:'test',b:1,c:true};
for(var i in obj){
 console.log(i,obj[i]);
}
console.log(obj);
a test
b 1
c true
{a:'test',b:1,c:true}

in 不仅可以用来遍历对象, 还可以用来遍历数组,不过i 对应与数组的key值

var arr =['1','test',2,true,'123dfg'];
